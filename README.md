# Game API
Olá, esse é o repositório da Game API, aqui você vai encontrar um exemplo de como documentar um serviço web utilizando o framework Swagger.

## Tecnologias Implementadas
* ASP.NET Core 2.1.4
* Visual Studio Code
* MacOS
* Swashbuckle.AspNetCore.Swagger
* Git e GitHub
